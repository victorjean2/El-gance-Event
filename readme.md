# Élégance Event

## Présentation
Élégance Event est un projet de site web réalisé dans le cadre de la formation DWWM.

## Objectif
Permettre à un utilisateur de :
- créer un compte
- se connecter
- se déconnecter
- demander un devis
- passer une commande

## Technologies
- HTML
- CSS
- JavaScript
- PHP
- MySQL
